## 组织结构

> 每一部分内容会有专门的`readme`解释

插件主目录均以小写命名，本文档只提供中文说明；

- 基本组织结构：

|Path|Name|Note|
|:-:|:-:|:-:|
|libs|库|第三方依赖库|
|locales|本地化|插件本地化|
|theme|主题扩展资源|插件本地资源|
|extend|第三方扩展|其他插件扩展接口|
